# Dodgers Tracker

Fanowska strona internetowa śledząca drużynę Los Angeles Dodgers z MLB. Wyświetla aktualne mecze, wyniki przeszłych spotkań, formę drużyny oraz tabele ligowe, korzystając z oficjalnego API MLB.

## Opis projektu

Dodgers Tracker to fanowska strona internetowa poświęcona drużynie Los Angeles Dodgers z Major League Baseball (MLB). Projekt umożliwia śledzenie aktualnych meczów, wyników przeszłych spotkań, formy drużyny oraz tabeli ligowej. Strona korzysta z oficjalnego API MLB do pobierania danych w czasie rzeczywistym.

**Uwaga:** To jest projekt fanowski i nie jest powiązany z MLB ani nie ma charakteru komercyjnego.

## Funkcje

- **Aktualny mecz:** Wyświetla informacje o trwającym meczu Dodgers lub najbliższym zaplanowanym spotkaniu, wraz z odliczaniem czasu do rozpoczęcia.
- **Przeszłe mecze:** Pokazuje ostatnie zakończone mecze z wynikami, datami i miejscami rozgrywek.
- **Starsze mecze:** Lista wcześniejszych spotkań z możliwością ładowania kolejnych wyników.
- **Forma drużyny:** Wizualizacja ostatnich 5 meczów Dodgers w formie ikon (wygrana/przegrana/remis).
- **Tabela ligowa:** Ranking drużyn MLB, AL i NL z możliwością filtrowania.
- **Przełącznik motywu:** Zmiana między ciemnym a jasnym motywem strony.
- **Linki do oficjalnych stron:** Szybki dostęp do stron MLB i Dodgers.
